## General prerequisites

Before you begin any chapter:

1. Install **VS Code**.
2. Ensure you can access this repository (provided by your facilitator).
3. Create a working folder for notes (recommended: inside the relevant chapter’s `Labs/` folder).

- Commit early and often.
